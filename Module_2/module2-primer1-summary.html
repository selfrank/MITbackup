Array

An array is an ordered collection of data. Compared to a variable, that can only store one value, arrays are used to store multiple values in a single variable.

Each item in an array has a number attached to it called the numeric index, which can be accessed. In JavaScript, arrays start at index zero and can be manipulated by various methods. 

var carBrands = ["honda", "mitsubishi", "bmw"];
var numbers = [1, 3, 6, 98];

Object

One of JavaScript's data types, an object is used to store various collections and more complex data entities using s key/value pairs. In a key/value pair, the first data entity is called the key and the second data entity is called the value. For example, in the key/value pair: {name: "Steve"}, the key is "name", and the value is "Steve". Anything can be stored in a key/value pair, ranging from arrays, numbers, strings, to even other objects!

Here is a more robust example: 

var dreamCar = {
  brand: "Audi";
  model: "A6";
  cylinders: 6;
}

The keys in the dreamCar object are brand, model, and cylinders. What values do each key point to?

String

A string is a sequence of characters used to represent text. In JavaScript, a string is one of the primitive values.

var myName = "Steven";
var favoriteCity = "New York";

setTimeout()

The setTimeout() method calls a function or evaluates an expression after a specified number of milliseconds.

setTimeout(myFunction, 3000);

setInterval()

The setInterval() method calls a function or evaluates an expression at specified intervals (in milliseconds).

The setInterval() method will continue calling the function until clearInterval() is called, or the window is closed.

setInterval(callAgain, 3000);

Conditionals

Conditionals are blocks of code that execute depending on if an expression evaluates to true. Some examples of conditional statements are if/then, while, and do/while.

State

State is the status particular object or piece of data, which can change (for example based on time or location). For an object to be stateful, it needs to have the ability to remember its previous status.

Variables

Variables are used to hold data. There are three different types of variable declarations in JavaScript, namely var, let, and const.

HTML Elements

HTML elements are tags that tell your browser window what sort of HTML elements to create on the webpage. These tags wrap content. Common examples are <h1>, <p>, <img>, <link>, <a>, <li> and <ol>.

Basic Styling Components
